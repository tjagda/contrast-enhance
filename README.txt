Description
********************
Attempt to enhance contrast and calculate the success through the Universal
Quality Index (UQI)


How to run
********************
The code runs on Python 3.6. The required packages are:
- numpy
- math
- scipy
- pylab
- cv2 (opencv)
- sewar

To run the files, just simply enter the following command in the terminal:
python3 test.py

To run the test with UQI scores, remove all pictures from the original_images
folder temporarily that don't start with "eg2_" and then uncomment the marked
sections before running the above command.
